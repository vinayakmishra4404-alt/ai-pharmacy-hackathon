# AI-Driven Pharmacy Access Platform

Hackathon project by **CODE SHARKS – VIT Bhopal**

## Problem
- Medicine shortages
- Expiry-based wastage
- Prescription misinterpretation

## Solution
An AI-powered platform that predicts demand, reduces waste,
and improves medicine accessibility.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (Netlify)
- Backend: Python (Flask / FastAPI)
- AI Modules: NLP, Demand Prediction, Rule Validation

## Architecture
Frontend → API Gateway → AI Intelligence → Data Layer → Pharmacies

## Deployment
- Frontend: Netlify
- Backend: To be connected via `main.py`
